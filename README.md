# DOM Assignment 3

## How to run

1. Open the folder in VS Code.
2. Open `index.html`.
3. Use Live Server or open the file in a browser.

## What this project does

- Shows a greeting at the top of the page.
- Lets the user click color boxes to fill and unfill them.
- Lets the user type a name and press Greet to update the heading.

## What I learned

- How to select HTML elements in JavaScript.
- How to add click events to buttons.
- How to change text on the page using DOM manipulation.
- How to change styles from JavaScript.
